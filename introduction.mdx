## Overview

Pavel Melnik empowers teams to manage projects efficiently with a clean, intuitive interface. You create tasks, assign roles, track progress, and collaborate in real-time without the clutter of traditional tools. Built for modern workflows, it integrates seamlessly with your existing stack to boost productivity.

Whether you run sprints, handle client deliverables, or coordinate cross-functional teams, Pavel Melnik simplifies everything. Focus on results, not tool overhead.

<Callout kind="info">
Pavel Melnik supports unlimited projects on free plans for small teams.
</Callout>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com` and create your free account.
  </Step>
  <Step title="Create a Project" icon="plus">
    Click "New Project" and set your team name and goals.
  </Step>
  <Step title="Invite Team Members" icon="mail">
    Add collaborators via email. They join instantly.
  </Step>
  <Step title="Add Your First Task" icon="plus-circle">
    Use the quick-add form: title, assignee, due date.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Fetch your projects via API
  const response = await fetch('https://api.example.com/v1/projects', {
    headers: {
      Authorization: `Bearer ${YOUR_API_KEY}`
    }
  });
  const projects = await response.json();
  console.log(projects);
  ```
  ```python
  # Fetch projects with Python
  import requests
  response = requests.get(
    'https://api.example.com/v1/projects',
    headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
  )
  projects = response.json()
  print(projects)
  ```
</CodeGroup>
