---
title: Can't Upload Files
description: Solve file upload issues in Pavel Melnik.
---

## Symptoms

- Upload button is disabled
- Error: "File upload failed"

## Causes

- Unsupported file type
- File size exceeds 100MB limit

## Quick Fix

<Steps>
  <Step title="Check file type">
    Ensure your file is PDF, DOCX, PNG, or JPG.
  </Step>
  <Step title="Reduce file size">
    Compress files to below 100MB.
  </Step>
  <Step title="Refresh page">
    Reload Pavel Melnik and try again.
  </Step>
</Steps>
