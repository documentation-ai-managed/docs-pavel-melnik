---
title: Create a New Project
description: Start managing a project with Pavel Melnik.
---

## Steps

<Steps>
  <Step title="Open Project Dashboard">
    Click "Projects" in the main menu.
  </Step>
  <Step title="Click 'New Project'">
    Press the blue "New Project" button.
  </Step>
  <Step title="Fill details">
    Enter project name, description, and set a deadline.
  </Step>
  <Step title="Add team members">
    Assign roles to your teammates.
  </Step>
  <Step title="Save and start tracking">
    Click "Create" to begin managing your project.
  </Step>
</Steps>
