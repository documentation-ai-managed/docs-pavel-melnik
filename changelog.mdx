## v1.3.0 (2024-11-01)

<Update label="2024-11-01" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time task comments with @mentions for better team collaboration
- Introduced custom workflows with drag-and-drop automation builder
- New integrations with Slack and Google Workspace for seamless notifications

## Improvements

- Enhanced dashboard performance by `>40%` with optimized data fetching
- Improved mobile responsiveness for task boards on all devices

## Bug Fixes

- Fixed sync issues when assigning tasks to multiple team members
- Resolved notification delays in high-traffic projects

To upgrade, follow these steps:

<Steps>
  <Step title="Backup Data" icon="database">
    Export your projects via the dashboard.
  </Step>
  <Step title="Update CLI" icon="download">
````bash
npm install -g @pavelmelnik/cli@latest
````
  </Step>
  <Step title="Run Migration" icon="settings">
````bash
pavelmelnik migrate
````
  </Step>
</Steps>

</Update>

## v1.2.0 (2024-10-15)

## v1.1.0 (2024-09-20)

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "breaking"]}>

## New Features

- Time tracking integrated directly into tasks
- Role-based permissions for teams (Admin, Editor, Viewer)
- Webhook support for custom integrations

## Breaking Changes

- Updated authentication to use JWT tokens instead of API keys for new endpoints
- Migrate your apps using the new auth flow

## Bug Fixes

- Patched security vulnerability in project sharing links
- Improved error handling for invalid date inputs

<Callout kind="alert">
  If you use legacy API keys, update to JWT before November 1, 2024.
</Callout>

</Update>

## How to Stay Updated

Enable auto-updates in your Pavel Melnik dashboard settings or use the CLI command:

````bash
pavelmelnik check-updates
````

For full release notes and migration guides, visit the [dashboard](https://dashboard.example.com/releases).

<Tabs>
  <Tab title="CLI" icon="terminal">

  Monitor updates via CLI:

  ````bash
  pavelmelnik status --updates
  ````

  </Tab>
  <Tab title="Dashboard" icon="monitor">

  Check the updates tab in your project settings.

  </Tab>
</Tabs>
